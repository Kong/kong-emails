Kong Enterprise 0.36 Release
This is an internal announcement. Customer outbound announcement will be released separately.


Greetings, Kongers!

We are proud to announce that Kong Enterprise 0.36 has been released and is now available on Bintray. 


This release rebases Kong Enterprise onto Kong 1.2.1.  Checkout the changelog below for details and links to the corresponding documentation.


Changelog
  • Adds Kong 1.1 - 1.2.1 Updates including:
    •  Bulk database imports
    • Cache Warming
    • Tags
  • Bundles the Mutual TLS Authentication plugin
  • Various updates to the Request Validator, OAuth2 Introspection, Rate Limiting Advanced, and Collector plugins

Read the complete changelog here: https://docs.konghq.com/enterprise/changelog/


  • The 0.35 to 0.36 Migration Guide is currently in review and can be seen here: https://github.com/Kong/docs.konghq.com-private/pull/304

  • Free Trials On-Prem 0.36 will be released shortly, a separate email will be sent when it is released.
